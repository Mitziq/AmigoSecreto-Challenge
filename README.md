<h2> :pushpin: Descripción del proyecto </h2>
<p1> El juego del amigo secreto es una aplicación web diseñada para que interactues con tus amigos y pases momentos divertidos con ellos. </p1>

<p2> :star2: Funcionalidades: </p2>


<p3> `1`: Verificar que no existan espacios vacios. 

`2`: Agregar nombres de los participantes. 

`3`: Asegurarse de que no se repitan los nombres. 

`4`: Asegurarse de que solo se ingresen nombres validos.

`5`: Verificar que no se repitan los nombres.

`6`: Reiniciar juego.
</p3>
